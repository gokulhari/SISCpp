Detailed Description

An example with a fourth order system.

Solves a fourth order system:

\[ \partial_{yyyy}u(y) \,-\, 4\,\partial_{yy}u(y) \,+\, 8\,u(y) \;= \; 1 \,-\, y^2 \,+\, y^3, \]

with clamped boundary conditions:

\[ \begin{array}{cll} \partial_yu(1) \;&=\; u(1) \;&=\; 0 \\ \partial_yu(-1) \;&=\; u(-1) \;&=\; 0. \end{array} \]
