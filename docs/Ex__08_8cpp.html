Detailed Description

Finds eigenvalues for the Orr-Sommerfeld operator.

Solving the Eigenvalue problem:

\[ \left(\frac{1}{Re}\partial_{yyyy} - \left(\frac{2}{Re} + (1-y^2)\,i \right) \partial_{yy} + \left(\frac{1}{Re} -2i + (1-y^2)\,i \right)\right)u \;=\; \lambda \; (\partial_{yy} - 1) u \]
